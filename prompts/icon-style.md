# 3D Icon Style System

## Style
cute, soft, rounded 3D illustration  
Toss-inspired design language  
extremely clean and minimal  

## Material
smooth matte plastic  
no texture  
no noise  
no gloss  

## Lighting
soft studio lighting  
very subtle shading  
no harsh highlights  

## Composition
single object only  
centered  
front-focused  
no perspective distortion  

## Depth
slightly thick  
soft rounded edges  

## Background
transparent or pure white only  

## Strict Rules
must be a single object  
must be centered  
must not include background elements  
must not include realistic textures  
must not include text unless essential  
must maintain consistent lighting  

## Negative
no shadow outside object  
no extra decoration  
no gradients on background  
no multiple objects
