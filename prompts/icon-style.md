# 3D Object Icon Style System

## Goal
Transform the uploaded reference image into a standalone 3D object icon.

## Style
cute, soft, rounded 3D illustration  
clean modern product icon style  
soft and friendly  
minimal but recognizable  

## Material
smooth matte plastic  
no texture  
no noise  
no gloss  
soft surface finish  

## Lighting
soft studio lighting  
subtle ambient shading  
soft depth  
no harsh highlights  

## Object Rules
the result must be a standalone object  
do not place the symbol on a square tile, app icon, badge, plate, or card  
do not embed the symbol into a background panel  
reconstruct the reference as the object itself  
preserve the core silhouette of the uploaded image  

## Shape Rules
rounded corners  
soft thickness  
simple, clean geometry  
slightly simplified details  
maintain recognizability  

## Composition
single object only  
centered  
large in frame  
isolated  
no extra scene elements  

## Background
transparent or pure white only  

## Strict Rules
must be a single standalone object  
must not include any base, stand, frame, tile, button, card, or container  
must not add decorative lines or bars  
must not create an app icon background  
must keep the uploaded image as the main shape reference  

## Negative
no square background  
no rounded square app tile  
no embossed symbol on a plate  
no extra decoration  
no multiple objects  
no realistic texture
